# NBWD Landing Pages

Shared landing page template system for New Bern Web Design client projects.

## Structure

- `index.html` — Base landing page template
- `css/style.css` — Core styles
- `js/main.js` — Interactions

## Workflow

1. Copy this repo for each new client landing page project
2. Customize the content placeholders in `index.html`
3. Push to GitHub — Vercel deploys automatically
4. Share the live URL

## Deployment

Connected to Vercel. Every push to `main` triggers a new deployment.
